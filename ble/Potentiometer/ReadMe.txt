Пример кода для компиляции в ARDUINO IDE

Работает по принципу ардуино, не подойдет  для компеляции в ESP IDF.

Идет имитация  работы по протоколу ДУТ ЭСКОРТ.
По ссылке можно посмотреть пример работы: https://rutube.ru/video/33e9a4885fcabe935e4bd54113330ec9/

Для имитации работы емкостного датчика, трубок как в датчике уровня топлива, используется потенциометр, изменяя его сопративление мы меняем значение.

АЦП ESP32 12 битный и значение считываются от 0 до 4095.

Через функцию MAP() значение от 0 до 4095 конвектируется в значение от  0 до 100.

Значение от 0 до 4095 записывается в теги уровня топлива.

Значение от  0 до 100 записывается в тег датчика температуры.

В коде присуствует массив изменяя его можно создать свой формат без проводного датчика.
